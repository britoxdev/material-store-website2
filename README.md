# 🧱 Cabrália Construction Materials - Landing Page

Modern and responsive landing page developed for a construction materials store.

## 🚀 Features

- ✅ Responsive layout (mobile-first)
- 🌙 Dark mode toggle
- 📱 WhatsApp integration
- 🎨 Modern UI with Bootstrap 5
- 🧾 Contact form (redirects to WhatsApp)
- ⚡ Smooth user experience

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- Bootstrap Icons
